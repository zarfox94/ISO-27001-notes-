# Cybersecurity Knowledge Base

A structured collection of study notes and documentation related to cybersecurity, governance, and information security management systems.

This repository contains organized notes from my personal learning journey in cybersecurity, including ISO/IEC 27001, auditing processes, risk management methodologies, and security best practices.

The goal of this repository is to transform raw study notes into clear, structured technical documentation that can be used both as a personal reference and as a public knowledge base.

---

# Topics Covered

## ISO/IEC 27001 – Information Security Management Systems (ISMS)

Documentation and notes covering the structure, implementation, and auditing of ISO/IEC 27001.

Topics include:

* Overview of ISO/IEC 27001
* ISMS framework and objectives
* PDCA (Plan–Do–Check–Act) model
* Risk assessment and risk treatment
* Statement of Applicability (SoA)
* Internal audit processes
* Certification lifecycle
* Surveillance and recertification audits
* Nonconformities and corrective actions
* Normative vs informative criteria
* ISO 27002 implementation guidance
* Annex A security controls
